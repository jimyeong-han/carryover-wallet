# 💰 이월 가계부 (Carryover Wallet)

하루 사용 가능 금액(일일 예산)을 정하고, 쓰고 남은 잔액을 자동으로 **이월**해 생활비를 관리하는 가계부 PWA입니다. 매달 초 자동으로 초기화됩니다.

## 핵심 로직

```
현재 잔액 = 일일예산 × 이번 달 지난 일수 − 이번 달 총 사용액
```

덜 쓴 날의 금액이 다음 날 잔액으로 자동 누적(이월)됩니다.

## 주요 기능

- 일일 예산 수동 설정 (⚙ 설정)
- 지출 기록 (금액 · 날짜 지정 · 카테고리 · 메모)
- 이월 잔액 · 오늘 남은 몫 · 사용률 표시
- 매월 자동 초기화 (지난 달은 보관함에 요약 저장)
- PWA — 홈 화면 설치 · 오프라인 동작
- 데이터는 브라우저 localStorage에 저장 (기기 내 저장)

## 사용법

정적 파일이라 별도 빌드가 필요 없습니다.

- 로컬 실행: `python -m http.server 8080` 후 `http://127.0.0.1:8080`
- 모바일: GitHub Pages 등 HTTPS 주소로 접속 → Chrome "홈 화면에 추가"
